# Разбор архитектуры платформы VETVI

Здесь ты можешь расписать проект максимально подробно.

### Технические фичи бэкенда:
* **FastAPI + SQLAlchemy 2.0:** Использование асинхронного движка для оптимизации пула соединений с PostgreSQL.
* **Безопасность:** Реализация авторизации с использованием асимметричного шифрования JWT RS256.
* **Миграции данных:** Настройка Alembic для сложной структуры связей генеалогического древа.
